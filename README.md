# model-helper

## System Requirement
	Windows platform
	SAS Foundation needs to be installed
	Django 1.8
	Python 3+
	Database used sqlite3
	
## Features
	Import SAS spk model
	Scoring SAS model
	Generate Lift(ROC,Lift and Gain) report for models
	
## Used Modules
	pip install sas7bdat
	pip install django-jquery
